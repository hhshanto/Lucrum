Add a Walmart store

The client secret is encrypted with your WALMART_SECRET_KEY before it's saved — the plain secret is never stored. You're granted admin on the new store automatically, so it shows up in your store picker right away. Get the credentials from Walmart Seller Center → Settings → API keys.

Add or update a teammate

Creates the login if it doesn't exist yet, then grants it a role on the selected store. Admin can fix and delete entered rows; member can only read and add — use member for anyone logging purchases, since deleting a purchase silently changes your profit. If the email already has a login, the password is ignored and only the role is applied.

Who has access
